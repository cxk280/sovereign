# ide

Developer-facing tooling backed by the internal, OpenAI-compatible [`gateway`](../gateway) endpoint.

- **`sov` CLI** — chat, review a diff, generate tests from the terminal.
- **Continue.dev / Cursor config** — point an existing IDE at the private endpoint (drop-in).
- **Minimal VS Code extension** — commands wired to the gateway.

_Built in build-order Step 6._
