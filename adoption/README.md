# adoption

Measure whether the platform is actually helping — the "drive adoption and measure impact" surface.

- Aggregates gateway usage (requests, tokens, latency) and acceptance signals from the IDE/CI.
- Produces an estimated time-saved / impact report and feeds the dashboard's adoption view.
- Ships a **Grafana** dashboard JSON alongside the Prometheus metrics the gateway emits.

_Built in build-order Step 6._
